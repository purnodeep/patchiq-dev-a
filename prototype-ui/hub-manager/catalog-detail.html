Description

Critical security update addressing remote code execution vulnerabilities in Windows Server 2022, Windows 11, and Windows 10. Patches a Windows Kernel Elevation of Privilege Vulnerability (CVE-2024-21307) and related flaws that allow authenticated local attackers to execute arbitrary code with SYSTEM privileges.

🛡 Remediation Notes

Deployment recommendation: Deploy immediately to all Windows Server 2022 and Windows 11 systems. This update requires a system restart.

Pre-requisites: Requires Windows Server 2022 RTM or later. Ensure KB5028171 is installed before applying this update.

Known issues: Some environments using Windows Local Administrator Password Solution (LAPS) may encounter issues. Apply hotfix KB5034123 if affected.
