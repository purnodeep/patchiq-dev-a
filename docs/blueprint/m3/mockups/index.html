Design System

Design Principles

Five Rules. No Exceptions.

Every design decision in these mockups traces back to one of five principles.

01

Color = Signal

95% grayscale. Emerald = healthy, Red = critical, Amber = warning. Color only appears when it means something.

02

Precision Typography

Geist for display, GeistMono for data. A 4-level hierarchy enforced across every surface in the system.

03

True Black Base

#000 dark, #f5f5f5 light. Cards float via border and shadow — not via elevated gray backgrounds.

04

Monochrome Tags

No rainbow. No colored badges. Text content differentiates tags — not color. All tags look identical.

05

Data Viz in Forms

Every form shows live impact visualization. Users see what changes before they confirm.

Design Decisions

Canonical Reference

Every mockup adheres to these decisions. No exceptions without a documented reason.

Theme System
Dark default: #000 page, #111113 cards, #222 borders
Light mode: #f5f5f5 page, #fff cards, #e0e0e0 borders
Default accents: Emerald (dark), Purple (light)
8 curated accent colors + custom hex input
Signal colors are theme-invariant — same hex in both modes
healthy
critical
warning
Typography
Geist Variable — display + body text
GeistMono Variable — data, IDs, code, numbers
7-level scale: Display → Title → Heading → Body → Small → Caption → Data
Letter-spacing: -0.03em titles, +0.08em labels
Tabular numerals via font-variant-numeric: tabular-nums
Color Rules
Signal: #22c55e healthy, #ef4444 critical, #f59e0b warning
Severity: text color ONLY. No badge backgrounds ever.
Tags: monochrome. ALL same visual style regardless of key:value.
Charts: single accent color at varying opacities (0.8 → 0.1)
Interactive states via border-color shift, not background fill
Layout + Components
Sidebar: 220px full / 48px rail / hidden (mobile)
Agent: 48px icon rail — always dark regardless of theme
Sheets: 560px right-side overlay, backdrop blur
Breakpoints: 1280 / 1024 / 768px
Toasts: bottom-right, 380px wide, colored dot + duration bar
Table density: 48px default / 36px compact (toggle per user)
