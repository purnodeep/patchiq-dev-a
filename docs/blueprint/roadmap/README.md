# PatchIQ Roadmap — Feature Documents

Feature-level documentation for planned capabilities beyond M2. Each document describes the vision, user value, architectural foundations, and dependencies.

All features are gated by RBAC permissions and license tiers (COMMUNITY/PROFESSIONAL/ENTERPRISE/MSP).

## Features

| Feature | Status | Milestone | Dependencies |
|---------|--------|-----------|-------------|
| [Compliance Engine v2](compliance-engine-v2.md) | Planned | M3 | Tags, run_script, workflow triggers |
| [Extended Agent Collectors](extended-agent-collectors.md) | Planned | M3 | Module registry, run_scan enhancements |
| [Development Compliance](development-compliance.md) | Planned | M4 | Compliance v2, script library |
| [Remote Access (RDP)](remote-access.md) | Planned | M4 | Agent system module, RBAC |
| [Alert Pipelines](alert-pipelines.md) | Planned | M3 | Workflow triggers, notification node, tags |
| [Script-based Custom Collectors](script-based-collectors.md) | Planned | M3 | Executor module, script library |
